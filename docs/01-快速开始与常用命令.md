# 01 快速开始与常用命令

目标：10 分钟内跑通番禺场景。

## 0. 推荐方式：YAML 配置启动

现在所有运行脚本都支持 `--config <yaml路径>`。  
建议先复制 `Framework/configs/` 下示例，再按需改参数。

内置示例配置：

- `Framework/configs/random_baseline.yaml`
- `Framework/configs/panyu_json_baseline.yaml`
- `Framework/configs/panyu_processed_baseline.yaml`
- `Framework/configs/experiment_matrix.yaml`

## 1. 环境准备

```bash
conda create -n DSHW python=3.10 -y
conda install -n DSHW -y -c conda-forge pyrosm geopandas pyarrow osmium-tool pyyaml
```

说明：仓库已包含 `Map Resource/panyu.osm.pbf`，可直接使用。

## 2. 番禺数据预处理

```bash
conda run --no-capture-output -n DSHW python "Map Resource/scripts/preprocess_panyu_pyrosm.py" \
  --pbf "Map Resource/panyu.osm.pbf" \
  --out-dir "Map Resource/processed/panyu"
```

输出：

- `Map Resource/processed/panyu/nodes.parquet`
- `Map Resource/processed/panyu/edges.parquet`
- `Map Resource/processed/panyu/stations.parquet`
- `Map Resource/processed/panyu/meta.json`

## 3. 跑基线仿真（推荐）

YAML 方式（推荐）：

```bash
python -m Framework.examples.run_panyu_processed_baseline \
  --config "Framework/configs/panyu_processed_baseline.yaml"
```

命令行方式（临时覆盖）：

```bash
python -m Framework.examples.run_panyu_processed_baseline \
  --processed-dir "Map Resource/processed/panyu" \
  --scheduler nearest \
  --vehicles 10 --tasks 120 --horizon 360
```

## 4. 跑对比实验矩阵（报告用）

YAML 方式：

```bash
python -m Framework.examples.run_experiment_matrix \
  --config "Framework/configs/experiment_matrix.yaml"
```

命令行方式：

```bash
python -m Framework.examples.run_experiment_matrix \
  --processed-dir "Map Resource/processed/panyu" \
  --out "Framework/output/experiment_matrix"
```

输出：

- `Framework/output/experiment_matrix/summary.json`
- `Framework/output/experiment_matrix/summary.csv`
- `Framework/output/experiment_matrix/summary.md`

## 5. 常用补充命令

随机基线（YAML）：

```bash
python -m Framework.examples.run_baseline \
  --config "Framework/configs/random_baseline.yaml"
```

番禺 json 图基线（YAML）：

```bash
python -m Framework.examples.run_panyu_baseline \
  --config "Framework/configs/panyu_json_baseline.yaml"
```

可选：若你有广东全量包，重新抽取番禺子集：

```bash
conda run --no-capture-output -n DSHW python "Map Resource/scripts/extract_panyu_pbf.py" \
  --src "<你的广东省原始osm.pbf路径>" \
  --out "Map Resource/panyu.osm.pbf"
```

可选：生成 json 图（兼容旧流程）：

```bash
conda run --no-capture-output -n DSHW python "Map Resource/scripts/build_panyu_graph.py" \
  --pbf "Map Resource/panyu.osm.pbf" \
  --out "Map Resource/analysis/panyu_graph.json"
```
